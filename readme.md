# Run styleguide

`npm install` follow by `npm start`

Access the styleguide at [http://localhost:3000](http://localhost:3000)


# Usage

To include in Zanata:

```sh
npm i zanata-ui --save
```

To include specific component:

```js
import { Button, Input } from 'zanata-ui'
